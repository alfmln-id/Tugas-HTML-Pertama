Definisi HTML

HTML merupakan bahasa pemrograman (HyperText Markup Language) sebenarnya adalah bahasa marka standar untuk membangun halaman web. Berkas ini tersusun dari berbagai tag yang diapit simbol < dan >. Sebagai contoh: <html>, <head>, dan <body>.

File dengan ekstensi .html bisa dibuat melalui Text Editor sederhana. Namun, untuk pengerjaan proyek profesional seperti ALF MLN STORE, penggunaan editor khusus sangat disarankan untuk meminimalisir kesalahan penulisan kode.
